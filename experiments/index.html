Key Findings

  • Text prompts work! No need for expensive character reference APIs
  • Flux Dev = Best quality for final production ($0.03/img)
  • Flux Schnell = Best value for iteration ($0.003/img)
  • Ideogram = Great for bold/graphic style ($0.02/img)
  • Key: Include detailed character description + consistent style in EVERY prompt

Winning Prompt Formula

CHARACTER = """A small bright lime-green gecko with big friendly eyes
with white highlights, yellow belly, striped curly tail, four stubby
legs, cute and cartoonish, standing upright"""

STYLE = """childrens book illustration, simple flat cartoon style,
bold black outlines, bright saturated colors, clean digital art, no text"""

prompt = f"{CHARACTER}, {scene_description}, {STYLE}"
